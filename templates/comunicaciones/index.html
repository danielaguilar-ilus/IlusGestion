Twilio — Configuración
Modo Sandbox Twilio: el destinatario debe enviar primero join <código> al número sandbox +1 415 523 8886 para recibir mensajes.
Empieza con "AC". Puedes pegar la URL completa de Twilio.
¿Cómo funciona?
  1. Crea una cuenta en twilio.com
  2. Ve a Messaging → Try it out → Send a WhatsApp message
  3. El destinatario envía el código join al +1 415 523 8886
  4. Ingresa aquí tu Account SID y Auth Token desde el Dashboard
  5. Usa +14155238886 como número From en sandbox
  6. Para producción: verifica tu número en Twilio y actualiza el From
Asegúrate de tener instalado pip install twilio en el servidor.
